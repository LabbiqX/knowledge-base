## TL;DR
Alexander Embiricos, head of OpenAI's Codex, says the "current underappreciated limiting factor" to AGI is human typing speed — because humans still need to write prompts and validate AI's work. The solution: rebuild systems so agents are "default useful" without requiring constant human prompting and review. He predicts early adopters will see "hockey stick" productivity gains starting 2026, followed by larger companies, with AGI arriving somewhere in that progression.

---

## Key Points
- Human typing/multitasking speed is the limiting factor to AGI, not model capability
- Current bottleneck: humans must review all AI-generated code and write prompts
- Solution: let agents validate their own work (agent-on-agent review)
- "If we can rebuild systems to let the agent be default useful, we'll start unlocking hockey sticks"
- Each use case needs its own approach — no simple path to full automation
- Prediction: early adopters hockey-stick productivity in 2026, larger companies follow
- AGI will arrive when productivity gains flow back into AI labs themselves
