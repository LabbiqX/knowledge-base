## TL;DR
A comprehensive open-source collection of 50+ Claude Skills is available on GitHub (ComposioHQ/awesome-claude-skills) under Apache 2.0 license. Skills span 9 categories: Document Processing, Development Tools, Data Analysis, Business, Communication, Creative Media, Productivity, Project Management, and Security. Each skill is a folder with a SKILL.md file.

---

## Key Points
- 50+ skills across 9 categories, Apache 2.0 licensed
- Can be added via Claude.ai settings, Claude Code config, or API
- Each skill = folder with SKILL.md file (essentially structured prompts)
- Skills becoming standard approach across AI platforms
- OpenAI also quietly adopted Anthropic's modular skills framework
- Best value comes from customizing skills to fit specific workflows
- Good source of inspiration even if not used directly
