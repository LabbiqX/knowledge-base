## TL;DR
Andrej Karpathy, who coined "vibe coding" in 2025, now declares it passé. LLMs have gotten smart enough that AI-assisted programming has evolved from fun throwaway projects into professional workflows. His new preferred term: "agentic engineering" — where developers orchestrate agents (not write code directly 99% of the time) while maintaining quality oversight.

---

## Key Points
- Karpathy coined "vibe coding" on Feb 2, 2025 as a "shower thoughts throwaway tweet" that minted a term at the right moment
- Original vibe coding: fun demos and explorations when LLM capability was lower
- One year later: LLM agents are becoming default professional workflow
- New term: "agentic engineering" — "agentic" because you orchestrate agents, "engineering" because there's art, science, and expertise
- Goal: claim agent leverage without compromising software quality
- 2026 will see improvements on both model layer and agent layer
- Agentic engineering is a learnable, improvable skill
