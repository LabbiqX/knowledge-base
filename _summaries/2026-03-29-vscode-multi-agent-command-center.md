## TL;DR
VS Code v1.109 introduces support for Anthropic Claude and OpenAI Codex agents alongside GitHub Copilot, transforming the editor into a multi-agent orchestration platform. Developers can now run multiple AI assistants side-by-side and delegate work based on each agent's strengths, shifting from single-agent to multi-agent workflows.

---

## Key Points
- VS Code v1.109 (Feb 4, 2026): Claude and Codex agents join GitHub Copilot
- Developers can delegate tasks to different agents based on their strengths
- Unified session management for orchestrating multiple AI assistants
- Claude integration via Anthropic's Agent SDK (public preview)
- Local agents for fast interactive help; cloud agents for longer-running tasks
- Microsoft's strategy: VS Code as Universal AI Interface rather than betting on one model
- "Microsoft's biggest strength isn't any single model but VS Code itself" — Futurum Group analyst
