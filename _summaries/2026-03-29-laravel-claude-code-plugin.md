## TL;DR
Taylor Otwell released a Laravel-focused Claude Code plugin (laravel/claude-code) that brings automated code cleanup to Laravel and PHP workflows. It's a port of the internal code-simplifier agent open-sourced by Boris Cherny. The plugin focuses on reducing complexity, improving readability, and cleaning up messy diffs after long coding sessions.

---

## Key Points
- Created by Taylor Otwell, Laravel creator
- Port of Claude Code team's internal code-simplifier agent
- Install via: `/plugin marketplace add laravel/claude-code`
- Run: "Review recent changes using the laravel-simplifier agent"
- Focuses on code quality without changing behavior
- Ideal for post-session cleanup and pre-PR review
- Adapted specifically for Laravel and PHP conventions
