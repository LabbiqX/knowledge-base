## TL;DR
Claude Cowork = "Claude Code for the rest of your work." CLI tabanlı Claude Code'un agentic yeteneklerini daha geniş kullanıcı kitlesine açıyor. Claude Code $1B yıllık gelire ulaştı (Anthropic'in ~%15'i). DNA analizi, vergi dosyalama, domates yetiştirme gibi yaratıcı kullanımlar ortaya çıktı. Cowork iki haftada, çoğunlukla Claude Code tarafından inşa edildi. Opus 4.5 kodlamada güçlü ama genel bilgi işlerinde sınırlı — Remote Labor Index'te 240 projeden sadece 9'unu tamamladı.
