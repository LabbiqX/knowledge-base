## TL;DR
Figma'nın yeni Claude Code entegrasyonu (MCP üzerinden), production/staging/localhost'taki gerçek UI'ları tarayıcıdan yakalayıp düzenlenebilir Figma frame'lerine dönüştürüyor. Bu, kod-first workflow'larla başlayan işleri tasarım canvas'ına taşıyarak ekip işbirliğini, varyasyon keşfini ve daha erken karar almayı mümkün kılıyor.

---

## Key Points
- Claude Code'dan çıkan UI'ları Figma canvas'ına capture edip düzenlenebilir frame'lere çevirir
- Production, staging veya localhost — herhangi bir ortamdan çalışır
- Birden fazla ekranı tek session'da yakalayarak flow bağlamını korur
- Kod güçlü "converging" (tek duruma varma), canvas güçlü "diverging" (alternatifleri keşfetme)
- Varyasyonları kod yeniden yazmadan deneyebilme — duplicate, rearrange, compare
- Tasarımcı, mühendis ve PM'ler aynı artifact üzerinde aynı fidelity seviyesinde çalışır
- Figma Make ile prompt-first, Claude Code ile code-first — aynı hedefe farklı başlangıç noktaları
- MCP catalog üzerinden diğer agentic araçlarla da uyumlu

---

## 🎯 Anahtar Çıkarım
> Claude Code → Figma entegrasyonu, kod-first exploration'ın en büyük darboğazını (ekip işbirliği) çözerek, hızla inşa edileni kanvas üzerinde birlikte keşfedip şekillendirmeyi mümkün kılıyor.
