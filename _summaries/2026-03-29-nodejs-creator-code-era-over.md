## TL;DR
Node.js yaratıcısı Ryan Dahl, "insanların kod yazma çağının bittiğini" söylüyor. Mühendisleri gereksiz kılmadığını ama satır satır syntax yazma işinin artık önemini yitirdiğini belirtiyor. Gelecekte mühendisler sistem mimarisi, AI output review, doğrulama ve "ne inşa edilecek" kararlarına odaklanacak.

---

## Key Points
- "The era of humans writing code is over" — satır satır syntax yazma artık core iş değil
- Mühendisler gereksiz olmuyor ama rolleri evrimleşiyor
- Gelecek odak: sistem mimarisi, AI-generated code review, output doğrulama, "what to build" kararları
- Google ve Microsoft: production kodun ~%30'u AI tarafından üretiliyor
- Anthropic: Claude Code'un ~%80'i AI tarafından yazılmış
- Dahl: geçiş "disturbing" ama daha yüksek seviye işler var
- Geoffrey Hinton: AI 2026'da milyonlarca işi değiştirebilir — karmaşık reasoning dahil
