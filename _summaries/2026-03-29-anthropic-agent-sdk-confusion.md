## TL;DR
Anthropic updated its Claude Code documentation to prohibit third-party tools from using OAuth tokens from Pro/Max plans, causing community backlash. OpenClaw, NanoClaw, and similar personal AI agents use these tokens extensively. After the uproar, Anthropic clarified that personal use cases and experimentation are still permitted — the restriction targets commercial services routing requests through consumer plan credentials.

---

## Key Points
- Anthropic updated TOS to prohibit third-party developers from routing requests through Free/Pro/Max plan credentials
- This directly affected OpenClaw and NanoClaw's onboarding flow which relies on these tokens
- Community backlash erupted on Reddit, X/Twitter, and forums
- The Agent SDK is specifically designed to let users build AI agents with Claude Code at its core
- Anthropic clarified: personal use is still fine; the restriction targets commercial exploitation of consumer plans
- Tension between Anthropic's need to protect API revenue and fostering the agent ecosystem
