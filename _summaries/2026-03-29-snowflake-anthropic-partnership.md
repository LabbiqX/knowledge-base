## TL;DR
Snowflake and Anthropic announced a $200M multi-year partnership to bring Claude models directly into the Snowflake platform for 12,600+ global customers. The partnership enables AI agents that can perform complex multi-step analysis on enterprise data with >90% accuracy on text-to-SQL tasks. Claude is available via Snowflake Cortex AI across all three major clouds (AWS, GCP, Azure).

---

## Key Points
- $200M multi-year strategic partnership with joint go-to-market initiative
- Claude models available in Snowflake via Amazon Bedrock, Google Cloud Vertex AI, and Microsoft Azure
- Trillions of Claude tokens processed per month through Snowflake Cortex AI
- >90% accuracy on complex text-to-SQL tasks (Snowflake internal benchmarks)
- Snowflake Intelligence powered by Claude Sonnet 4.5 for natural language data queries
- Multimodal analysis: text, images, audio, tabular data — all via SQL
- Snowflake Cortex Agents for production-ready custom multi-agent solutions
- Snowflake internally uses Claude Code for developer productivity
- Focus on regulated industries: financial services, healthcare, life sciences
- Claude is the only frontier model available on all three major cloud providers
