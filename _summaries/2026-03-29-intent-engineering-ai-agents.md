## TL;DR
Cloudflare now supports real-time Markdown conversion at the network level for AI agents. This reflects a broader shift: the web must evolve to serve both humans and AI agents. However, modern web pages are designed as visual experiences with competing tensions — extracting "just the content" is non-trivial since spatial information, ads, and interactive elements are deeply intertwined.

---

## Key Points
- Cloudflare's network now supports real-time Markdown conversion for LLM agent requests
- Response includes x-markdown-tokens header showing token count for context window planning
- Modern web pages are complex visual experiences, not just text with irrelevant framing
- Spatial information on web pages cannot be transmitted sensibly in Markdown
- The web wasn't built for AI agents — same adaptation challenges as when it became a shop/identity system
- "Intent Engineering" and transparent web design are emerging as the future for human-agent coexistence
