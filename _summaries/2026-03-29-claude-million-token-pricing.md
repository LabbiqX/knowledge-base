## TL;DR
Anthropic removed the long-context pricing surcharge for Claude Opus 4.6 and Sonnet 4.6, making 1-million-token context windows available at standard per-token rates. Previously, requests exceeding ~200K tokens were billed at premium "long-context" rates. Now all requests use the same per-token rate regardless of prompt size.

---

## Key Points
- 1M token context window for Opus 4.6 and Sonnet 4.6 now generally available
- Long-context pricing surcharge (previously triggered at ~200K tokens) has been eliminated
- Same per-token rate applies regardless of prompt size
- Opus 4.6: enterprise workloads requiring sustained reasoning across large datasets
- Sonnet 4.6: efficient general-purpose model for high-throughput developer use
- Context window evolution: 200K → limited 1M access → public 1M (Sonnet 4, Aug 2025) → standard pricing
