## TL;DR
Claude Code's December 2025 update introduces Language Server Protocol (LSP) for real-time diagnostics, asynchronous sub-agents for parallel multitasking, Ultrathink Mode for maximum reasoning on complex problems, Slack integration for task delegation, and AutoCloud GUI with Kanban-style task management. Browser and mobile integrations enable coding anywhere, and reusable agent skills can be shared across teams.

---

## Key Points
- **LSP support:** Real-time diagnostics, go-to definitions, reference tracking (v2.0.55+)
- **Async sub-agents:** Execute multiple tasks simultaneously; share resources with main agent
- **Ultrathink Mode:** Maximum reasoning resources for complex algorithms and optimization
- **Context window tracking:** Real-time progress bar + `/stats` for session insights
- **Browser integration:** `/chrome` command for in-browser testing and debugging
- **Mobile support:** Android app for task management on the go
- **Slack integration:** Assign tasks directly from Slack to Claude Code
- **AutoCloud GUI:** Kanban-style standalone app for async task management
- **Expanded agent skills:** Open standard for reusable, task-specific modules
- **`altp` command:** Select models mid-prompt during extended sessions
