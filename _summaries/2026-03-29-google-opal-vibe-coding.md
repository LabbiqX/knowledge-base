## TL;DR
Google integrated its vibe-coding tool Opal into the Gemini web app, allowing users to create custom mini-apps (Gems) by describing them in natural language. Opal features a visual editor that turns prompts into step-by-step app construction without code. For advanced customization, users can switch to the Advanced Editor at opal.google.com.

---

## Key Points
- Opal now available in Gems manager within Gemini web app
- Visual editor: rearrange steps, link them together without writing code
- Natural language → step list → working mini-app
- Builds on Gems concept (customized Gemini for specific tasks)
- Advanced Editor at opal.google.com for deeper customization
- Competing with Lovable, Cursor, Anthropic, OpenAI in vibe-coding space
- Also competing with consumer-focused tools like Wabi
