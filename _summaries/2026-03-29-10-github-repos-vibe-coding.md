## TL;DR
A curated list of 10 GitHub repos for mastering vibe coding: Context Engineering Template (CLAUDE.md + PRP blueprints), Awesome Vibe Coding (ecosystem catalog), Vibe Coding Workflow (5-stage MVP process), Rulebook AI (portable packs for AI consistency), Claude Code Settings, AI Coding Style Guide (token compression), Vibe Check MCP (oversight server with 27% success rate improvement), Vibe Kanban (Rust-based multi-agent orchestration), and VibeKit (Docker sandbox safety layer).

## Key Points
- **Context Engineering Template:** CLAUDE.md, INITIAL.md, PRP blueprints for consistent AI coding
- **Awesome Vibe Coding:** Full ecosystem catalog from browser builders to terminal agents
- **Vibe Coding Workflow:** 5-stage process to turn ideas into MVPs in hours
- **Rulebook AI:** Versionable "Packs" syncing rules across Cursor, Gemini, Copilot
- **Claude Code Settings:** LiteLLM proxy, spec-driven commands (/specify, /plan, /implement)
- **AI Coding Style Guide:** 8-level compression reducing code to 20-50% of original size
- **Vibe Check MCP:** Chain-Pattern Interrupts preventing over-engineering (+27% success, -50% harmful actions)
- **Vibe Kanban:** Rust-based multi-agent orchestration platform
- **VibeKit:** Docker sandbox with automatic secret redaction, offline operation
