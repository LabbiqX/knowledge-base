## TL;DR
Google launched fully managed, remote MCP servers making its services (Maps, BigQuery, Compute Engine, Kubernetes Engine) plug-and-play for AI agents. Instead of weeks of connector setup, developers paste a URL to a managed endpoint. Google also leverages Apigee to translate standard APIs into MCP servers with existing security/governance controls.

---

## Key Points
- Initial MCP servers: Maps, BigQuery, Compute Engine, Kubernetes Engine
- "Making Google agent-ready by design" — Steren Giannini
- Works with any MCP client: Gemini CLI, AI Studio, Claude, ChatGPT
- No extra cost for existing enterprise customers during public preview
- Apigee integration: turn any API into MCP server with existing governance
- Security: Google Cloud IAM + Model Armor (anti-prompt injection, anti-data exfiltration)
- MCP donated to Linux Foundation for standardization
- More services rolling out weekly
