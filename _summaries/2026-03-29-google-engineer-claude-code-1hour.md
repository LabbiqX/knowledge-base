## TL;DR
Jaana Dogan, Google Principal Engineer on Gemini API, gave Claude Code a 3-paragraph problem description for distributed agent orchestrators and received a working result in one hour that matched what her Google team had been building for a year. She later clarified it was a toy version, not production-grade, but emphasized that once you have domain knowledge, rebuilding with AI is now trivial.

## Key Points
- Task: distributed agent orchestrators for coordinating multiple AI agents
- Only 3 paragraphs of prompt (simplified version, no internal details)
- Result wasn't perfect but matched year-long team effort conceptually
- Google only allows Claude Code for open-source, not internal projects
- Dogan's AI coding evolution timeline: 2022 lines → 2023 sections → 2024 multi-file → 2025 full codebases
