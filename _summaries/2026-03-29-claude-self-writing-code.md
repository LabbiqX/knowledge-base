## TL;DR
Anthropic CEO Dario Amodei, yeni Claude modellerini güçlendiren kodların %90'ından fazlasının Claude tarafından yazıldığını doğruladı. Bu, Mart 2025'teki "3-6 ay içinde AI kodların %90'ını yazacak" tahmininin gerçekleşmesi anlamına geliyor. Boris Cherny, Aralık 2025'teki tüm kod katkılarının tamamen Claude tarafından yazıldığını açıkladı.

---

## Key Points
- Claude, kendi halef modellerinin kodunun %90'ından fazlasını yazıyor
- Amodei'nin Mart 2025 tahmini doğrulandı
- Boris Cherny (Claude Code yaratıcısı): Aralık 2025'te tüm kod tamamen Claude'dan
- Claude Code'un kendi kod tabanının ~%90'ı AI tarafından oluşturuldu
- "Cowork" özelliği Claude Code ile ~10 günde inşa edildi
- Mühendisler hâlâ gerekli: düzenleme, en zor %10, AI gruplarının denetimi
- Güvenlik araştırmacıları: AI kodunun görülmemiş ölçekte güvenlik açıkları yaratabileceği endişesi
