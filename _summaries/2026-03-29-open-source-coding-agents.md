## TL;DR
Open-source coding agents (OpenCode, Cline, Aider) operate one layer above LLMs, keeping costs consistent by working across multiple models. OpenCode introduced a $10/month subscription (OpenCode Go). These tools mark the emergence of a new developer tooling layer where the agent — not the model — is the primary interface developers interact with.

---

## Key Points
- LLM economics breaking down: developers juggle multiple APIs with unpredictable token bills
- Open-source agents work across many LLMs, providing cost consistency
- OpenCode Go: $10/month subscription for managed workloads
- Key players: OpenCode, Kilo Code (16.3k stars), Cline (58.7k stars), Aider (41.6k stars)
- Value shifting from LLMs themselves to the agent layer above them
- Agents scan repos, interpret instructions, break tasks into steps, run commands, apply changes
- Subscription model becoming standard for packaging agent+model access
