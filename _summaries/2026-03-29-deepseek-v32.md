## TL;DR
DeepSeek introduces V3.2 and V3.2-Speciale, reasoning-first models with DeepSeek Sparse Attention (DSA) that reduces long-context inference cost by ~50% while maintaining quality. The 671B MoE model (37B active) achieves GPT-5-comparable performance, with Speciale reaching Gemini 3.0 Pro-level reasoning. Features include explicit thinking/non-thinking modes for tool use, 128K context window, and agent-native tool protocol trained on 85,000+ synthetic tasks across 1,800+ environments.

---

## Key Points
- DeepSeek Sparse Attention: O(kL) instead of O(L²) — ~50% lower long-context cost
- Lightning indexer + fine-grained selector keeps top-k key-value positions per query
- 671B total params, 37B active per token (MoE architecture)
- GRPO RL compute exceeds 10% of pre-training compute (massive for post-training)
- Specialist RL runs: math, competitive programming, reasoning, browsing/agents, safety
- 1,800+ synthetic environments, 85,000+ agent tasks for training
- Explicit thinking/non-thinking modes for tool calls
- V3.2-Speciale: gold medal at IMO 2025, CMO 2025, IOI 2025, competitive at ICPC Finals
- Open weights on HuggingFace
