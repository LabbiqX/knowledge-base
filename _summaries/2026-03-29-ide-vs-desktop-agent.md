## TL;DR
The IDE is no longer the center of software development. Three waves of AI coding tools have shifted the gravitational center: Wave 1 (AI as IDE feature — Copilot), Wave 2 (AI in terminal as execution layer — Claude Code, Gemini CLI), and Wave 3 (desktop agents that orchestrate work independently). Developers now spend more time delegating than coding.

---

## Key Points
- Wave 1: AI as IDE guest (autocomplete, inline edits, chat sidebars — GitHub Copilot)
- Wave 2: CLI agents accepting high-level instructions (Claude Code, Gemini CLI) — terminal becomes execution layer
- Wave 3: Desktop agents orchestrating work independently, IDE becomes verification/review tool
- Fundamental shift from "human writes code, tools assist" to "human orchestrates, agents execute"
- IDE still matters but is no longer the main stage — it's where software gets verified, not written
- Developer role shift: from bottleneck at every keystroke to orchestrator of coding agents
