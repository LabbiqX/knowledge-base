## TL;DR
Andrej Karpathy (ex-Tesla AI lead, OpenAI co-founder) reflected on the term "vibe coding" he coined in February 2025, predicting it will "terraform software and alter job descriptions." He describes vibe-coded software as "free, ephemeral, malleable, discardable after single use" — a fundamentally new type of code. He praised Claude Code as the "first convincing demonstration of what an LLM Agent looks like."

---

## Key Points
- Vibe coding empowers both non-professionals and trained professionals to write more software
- New code characteristics: free, ephemeral, malleable, discardable — fundamentally different from traditional software
- "Regular people benefit a lot more from LLMs compared to professionals, corporations and governments"
- METR study caveat: AI coding assistants decreased experienced developer productivity by 19%
- Non-technical workers building and selling apps in hours (Jack Dorsey vibe-coded BitChat)
- Claude Code praised as "first convincing demonstration" of an LLM Agent
- Karpathy coined term "totally oblivious to how far it would go"
- Called 2025 "an exciting and mildly surprising year of LLMs"
