## TL;DR
Perplexity launched Computer — a "general-purpose digital worker" that deploys subagents to browse, research, create, and connect tools automatically. It's like OpenClaw for normal people. Anthropic expanded Claude Cowork with 13 new enterprise plugins and Remote Control for mobile access. Notion released Custom Agents with programmable toolkits. The Pentagon chose OpenAI over Anthropic.

---

## Key Points
- Perplexity Computer: describe the outcome, subagents do the work (Gmail, Slack, Notion, Calendar integration)
- "Chat answers. Agents do tasks. Computer works." — runs hours or months
- Perplexity Computer = OpenClaw for non-technical users (browser-based, no setup)
- Anthropic: Claude Cowork + 13 enterprise plugins (Google Workspace, DocuSign, FactSet, LegalZoom)
- Claude Code gets Remote Control for mobile access + scheduling tool
- Notion Custom Agents: run workflows on schedule or on-demand, early-alpha toolkit on GitHub
- Anthropic released Skills repository on GitHub — full Agent Skills collection
