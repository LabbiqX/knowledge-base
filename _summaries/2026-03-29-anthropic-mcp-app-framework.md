## TL;DR
Anthropic released a major open extension to MCP that allows MCP servers to serve interactive app-like experiences within chat interfaces. Partners include Slack, Figma, Asana, Box, Canva, and others. The framework turns Claude into an app platform where users can preview documents, edit messages, and interact with third-party services directly in chat.

---

## Key Points
- MCP Apps is an open protocol extension — any chatbot provider can adopt it
- Already supported in Goose, VS Code (Insiders), and ChatGPT
- Early partners: Amplitude, Asana, Box, Canva, Clay, Figma, Hex, monday.com, Slack
- Box integration: search files, preview documents inline, ask questions about them
- Slack integration: write/edit message drafts and post directly from Claude
- Anthropic donated MCP protocol to Agentic AI Foundation in December
- This positions Claude as an app platform, not just a chatbot
