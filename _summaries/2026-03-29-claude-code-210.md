## TL;DR
Anthropic released Claude Code v2.1.0 with 1,096 commits introducing agent lifecycle hooks, hot-reloading skills, forked sub-agent contexts, wildcard tool permissions, session teleportation between terminal and web, and multilingual output. The update targets power users building autonomous workflows rather than casual AI assistants, reflecting Claude Code's transition from internal Anthropic tool to serious developer platform.

---

## Key Points
- **Agent lifecycle hooks:** PreToolUse, PostToolUse, Stop logic for fine-grained state management and audit logging
- **Hot reload for skills:** New/updated skills available immediately without restarting sessions
- **Forked sub-agent context:** `context: fork` in skill frontmatter for isolated execution
- **Wildcard tool permissions:** e.g., `Bash(npm *)` for easier access management
- **Session teleportation:** `/teleport` and `/remote-env` to move work between local terminal and claude.ai/code
- **Multilingual output:** Native language setting for global teams
- **Vim motions, /plan shortcut, real-time thinking display** and many UX improvements
- **Security fix:** Sensitive data (OAuth tokens, API keys) could be exposed in debug logs — now patched
- Community sentiment overwhelmingly positive; some calling it "closest thing to AGI"
