## TL;DR
Claude Code'un ileri düzey kullanımına odaklanan rehber, OpenAI ve Anthropic'teki mühendislerin artık kod yazmadığını, agent'ların yazdığını vurguluyor. Opus 4.6 ile 20 saatte sıfırdan C compiler inşa edilmesi, yazarın birkaç ay önce GitHub hesabı bile yokken şimdi yüzlerce commit attığı anlatılıyor. Skills, MCP'ler, hooks, memory yönetimi, GitHub workflow'ları ve persistent session'lar dahil 10 bölümlük ileri seviye rehber.

---

## Key Points
- OpenAI ve Anthropic mühendisleri artık kod yazmıyor; Codex/Claude yazıyor
- Boris Cherny (Claude Code lead): 40 mühendis 1.5+ yıl yerine, 5 mühendis 3-6 ay
- Opus 4.6 ile 20 saatte sıfırdan C compiler — "semantic knowledge of code" aşıldı
- Rehber konuları: setup, skills oluşturma/enjekte etme, MCP kullanımı, hooks, memory yönetimi, GitHub workflow, persistent sessions, release cadence
- Büyük ekipler, AI araçlarını derinlemesine bilen kişiler tarafından disrupted olacak
- Doğal dil programlama ve otonom çalışma yeni era
