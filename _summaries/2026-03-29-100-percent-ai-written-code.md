## TL;DR
Anthropic'in Claude Code lideri Boris Cherny, 2+ aydır hiç elle kod yazmadığını ve kodunun %100'ünü Claude Code + Opus 4.5 ile ürettiğini açıkladı. Günde 22-27 PR atıyor. OpenAI araştırmacısı Roon da aynı şeyi söyledi. Anthropic genelinde oran %70-90, Claude Code'un kendi kodunun %90'ı Claude Code tarafından yazılıyor.

---

## Key Points
- Boris Cherny: 2+ aydır elle kod yazmıyor, günde 22-27 PR
- Anthropic geneli: %70-90 AI-generated kod; Claude Code'un kendi kodu: %90
- OpenAI'da Roon: "Artık kod yazmıyorum, programlama her zaman çile çekmekti, bittiğine memnunum"
- Sektör ortalaması çok daha düşük: Microsoft %30, Salesforce %30, GitHub Python'da %29 (Science çalışması)
- Dario Amodei: 6-12 ay içinde AI tüm yazılım mühendisliğini yapabilir
- Cowork, Claude Code'un kendisi ile ~1.5 haftada inşa edildi
- Anthropic artık uzman yerine generalist işe alıyor
- Karpathy uyarısı: "subtle conceptual errors", dead code, over-complication
